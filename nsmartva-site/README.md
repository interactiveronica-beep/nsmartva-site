# Notary Smart Services Website

Static landing page for `www.nsmartva.com`.

## Files

- `index.html`: main page
- `styles.css`: page styling
- `assets/hero-office.png`: generated hero image
- `CNAME`: GitHub Pages custom domain setting

## Current Buttons

The buttons currently use email links:

- General requests: `assistant@nsmartva.com`
- Invoices/payment requests: `invoices@nsmartva.com`
- Process service/property management: `serves@nsmartva.com`
- Grants/nonprofit support: `grants@nsmartva.com`
- Veronica contact: `veronica@nsmartva.com`

These can be replaced with Microsoft Forms, Stripe, PayPal, or Square links later.
